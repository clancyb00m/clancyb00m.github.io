登录加密文件快速定位

方法1 密码框找id

通过开发者工具中的小箭头找到密码框，查找是否有 id 属性，如果有，例如 id=password，则通过 ctrl+shift+F 搜索 #password

方法2 登录按钮找绑定的事件

通过开发者工具中的小箭头点击登录按钮，定位到登录按钮的代码，查看是否有 onclick 事件，例如 onclick="login()" ， ctrl+shift+F 搜索 function login

方法3 burp抓包获取关键词

抓登录包，查看登录包中各个参数，提取关键词， ctrl+shift+F 搜索这些关键词，即可快速定位到加密文件以及函数

加密爆破

对参数加密分以下两种情况：

  1. 通过上一步搜索加密方法，如果是常规加密如 md5、 base64 、 sha1 等，可以直接使用burp suite Intruder模块中的 Payload Processing 添加加密方法

  2. 程序的加密方式是自己写的一套逻辑，则可以下载加密js文件，通过 jsEncrypter 插件或 Burpjsencrypter 插件对值进行加密
