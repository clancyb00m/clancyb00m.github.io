BurpSuite 之 Proxy

  1. Intercept Client Requests

在测试中经常会抓到一些不必要的数据包，比如我根本就不测百度，谷歌等，但是它总是会发送一些数据包，通过右键菜单如下设置，即可不再监听该host

在 options 菜单中的 Intercept Client Requests 选项中，可保存此次的配置，下次打开burp后直接 Load options ，避免测试时抓到一些不在测试范围内的包，比如临时邮箱平台、短信接码平台、谷歌相关的、shodan的api和53kf等一些请求包

BurpSuite 之 Intruder

  1. 四种爆破模式

    • Sniper

      • 狙击手模式，逐个爆破，给1个参数，数量100，就爆破100次，给两个参数，数量100，就爆破100+100次

    • Battering ram

      • 攻城锤模式，选两个参数，只给一个payload位，数量100，就同时爆破，参数值一样，100次

    • Pitchfork

      • 草叉模式，至少给两个参数，一一对应进行爆破

    • Cluster bomb

      • 集束炸弹，给两个参数，混合爆破，比如数量均为100，则爆破10000次

平时我用的最多的还是 Sniper模式 和 Cluster bomb模式

在 Payloads 菜单中，可对参数进行设置，比如爆破带有验证码的数据包或者加密爆破等，详见 https://clancyb00m.github.io/2020/06/28/验证码识别和爆破脚本分享与演示/
